# 正则资源篇

## 1️⃣ 常用正则表达式速查表

| 表达式               | 含义                 | 示例              |
| ------------------- | ------------------ | ----------------- |
| `[0-9]`             | 匹配数字             | `11546`           |
| `[1-9]`             | 匹配 1 到 9         | `5`               |
| `[a-z]`             | 匹配小写字母         | `sky` → 匹配 `s`  |
| `[A-Z]`             | 匹配大写字母         | `Sky` → 匹配 `S`  |
| `sky`               | 匹配指定单词         | 文本中含 `sky`    |
| `天空`                | 匹配指定汉字         | 文本中含 `天空`     |
| `[\u4e00-\u9fa5]`   | 匹配汉字             | `评价`            |
| `[ \f\r\t\n]`       | 匹配空白字符         | 空格、制表符等     |
| `\w`                | 匹配字母、数字、下划线 | `abc_123`         |
| `\W`                | 匹配非字母数字下划线   | `#@！`            |
| `\s`                | 匹配空白字符         | 空格、换行         |
| `\S`                | 匹配非空白字符       | 任意非空白文字     |

---

## 2️⃣ 重复匹配（量词）

| 表达式   | 含义                       | 示例                         |
| ------- | ------------------------ | ---------------------------- |
| `{n}`   | 精确匹配 n 次               | `10{2}` → `100`              |
| `{n,m}` | 匹配前一个字符 n 到 m 次      | `10{2,3}` → `100` 或 `1000`  |
| `{n,}`  | 匹配至少 n 次               | `10{2,}` → `1000`、`10000`... |
| `+`     | 匹配一次或多次              | `zo+` → `zo`、`zoo`          |
| `*`     | 匹配零次或多次              | `zo*` → `z`、`zo`、`zoo`      |
| `?`     | 匹配零次或一次              | `zo?` → `z`、`zo`            |

---

## 3️⃣ Web Scraper 中正则的局限性

- ❌ 不支持全局匹配（通常只返回第一个匹配结果）  
- ❌ 不支持忽略大小写标志（无法直接 `i`）  
- ❌ 不支持分组提取（无法用捕获组返回特定子组）  
- ❌ 不支持替换文本（仅用于简单提取）

> 这些限制会影响到复杂文本抽取场景，遇到复杂抽取建议把原始文本先导出到本地，再用 Python/JS 等工具做二次清洗。
